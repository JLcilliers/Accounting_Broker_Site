## Quality Standards

### Image Quality
- **Resolution**: 2x for Retina displays (@2x assets)
- **Compression**: Optimize without visible quality loss
- **Format**: Modern formats (WebP) with fallbacks (JPG/PNG)
- **Color Profile**: sRGB for web consistency
- **Max File Sizes**:
  - Hero image: 200KB
  - Section images: 150KB
  - Icons: 5KB (SVG), 10KB (PNG)
  - Logos: 20KB

### Icon Quality
- **Style Consistency**: All icons same style (line vs. solid)
- **Stroke Weight**: 2px consistent across line icons
- **Size**: Standardized sizing (48px, 64px, etc.)
- **Format**: SVG preferred for scalability
- **Color**: Use design system colors

### Typography Quality
- **File Format**: WOFF2 for modern browsers, WOFF for fallback
- **Subsetting**: Include only required characters/weights
- **Loading**: Font-display: swap to prevent FOIT
- **Weights**: Only include needed weights (400, 600, 700)

## Notes & Reminders

1. **Always maintain source files** (PSD, AI, Figma) for future edits
2. **Version control** all assets (use Git or cloud storage)
3. **Document sources** for all stock photography and icons
4. **Keep licenses** for all paid assets on file
5. **Test on multiple devices** before final approval
6. **Compress before deployment** but keep uncompressed originals
7. **Generate @2x versions** for all raster graphics
8. **Create sprite sheets** for frequently used small icons (optional optimization)
