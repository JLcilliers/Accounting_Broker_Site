## Design System Elements

### Spacing Reference
Use Canva's ruler/guides to maintain consistent spacing:
- Section padding: 80px top/bottom
- Element spacing: 24px (standard)
- Large gaps: 64px
- Tight spacing: 16px
- Micro spacing: 8px

### Shadow Presets
**Light Shadow (cards):**
- Offset: 0, 1px
- Blur: 3px
- Color: #000000, 10%

**Medium Shadow (hover):**
- Offset: 0, 4px
- Blur: 12px
- Color: #000000, 15%

**Strong Shadow (modals/popups):**
- Offset: 0, 8px
- Blur: 24px
- Color: #000000, 20%

### Border Radius Standards
- Cards: 12px
- Buttons: 8px
- Input fields: 6px
- Images: 12px
- Small elements: 4px

## Tips for Canva Efficiency

1. **Create Components**: Save button styles, card templates as reusable
2. **Use Grids**: Enable grid view for precise alignment
3. **Duplicate Smart**: Copy/paste sections to maintain consistency
4. **Layer Names**: Name layers clearly ("Hero - Headline", "CTA - Primary")
5. **Group Elements**: Group related items for easier management
6. **Save Versions**: Use Canva's version history
7. **Collaborate**: Share with team for feedback
8. **Brand Kit**: Add colors/fonts to Brand Kit for quick access
